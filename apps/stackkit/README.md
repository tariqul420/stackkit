# StackKit CLI

Create and enhance production-ready JavaScript applications with a single CLI.

## Install & Create

```bash
npx stackkit create my-app
```

This scaffolds a new project using opinionated, production-ready defaults.

## What you can build

* **Frameworks**: Next.js, React (Vite), Express
* **Languages**: TypeScript (default), JavaScript
* **Databases**: Prisma (PostgreSQL), Mongoose (MongoDB)
* **Auth**: Better Auth, Auth.js

## Enhance an existing project

```bash
npx stackkit add auth
npx stackkit add db
```

Add authentication or database support to an existing compatible project.

## Check project health

```bash
npx stackkit doctor
```

Validates project compatibility, environment variables, and installed modules.

## Quick start

```bash
npx stackkit create my-app
cd my-app
npm run dev
```

## Why StackKit

* Opinionated defaults for production
* Modular: create first, add features later
* Clean CLI with minimal configuration
* Works with existing projects
