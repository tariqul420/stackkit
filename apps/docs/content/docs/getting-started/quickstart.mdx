## Create a New Project

<Tabs items={['pnpm', 'npm', 'yarn', 'bun']}>
  <Tab value="pnpm">
    ```bash
    pnpm create stackkit-app my-app
    ```
  </Tab>
  <Tab value="npm">
    ```bash
    npx create-stackkit-app my-app
    ```
  </Tab>
  <Tab value="yarn">
    ```bash
    yarn create stackkit-app my-app
    ```
  </Tab>
  <Tab value="bun">
    ```bash
    bun create stackkit-app my-app
    ```
  </Tab>
</Tabs>

## Choose Your Stack

The CLI will guide you through an interactive setup:

```bash
✔ What framework would you like? › Next.js
✔ Choose database › Prisma (PostgreSQL)
✔ Add authentication? › Better Auth
✔ Install dependencies? › Yes

✓ Project created successfully!
```

## Start Building

<Tabs items={['pnpm', 'npm', 'yarn', 'bun']}>
  <Tab value="pnpm">
    ```bash
    cd my-app
    pnpm dev
    ```
  </Tab>
  <Tab value="npm">
    ```bash
    cd my-app
    npm run dev
    ```
  </Tab>
  <Tab value="yarn">
    ```bash
    cd my-app
    yarn dev
    ```
  </Tab>
  <Tab value="bun">
    ```bash
    cd my-app
    bun dev
    ```
  </Tab>
</Tabs>

Your app is running at [http://localhost:3000](http://localhost:3000)

## Next Steps

Now that your project is created:

1. **[Configure Authentication](/docs/modules/auth)** - Set up OAuth providers and customize auth flows
2. **[Set Up Database](/docs/modules/database)** - Connect to your database and run migrations
3. **[Add More Modules](/docs/cli/stackkit-cli)** - Add auth or database to existing projects

For detailed setup instructions, see the [Installation Guide](/docs/getting-started/installation).
