# Quick Start

Create your first StackKit project in under a minute.

## Create a New Project

<Tabs items={['pnpm', 'npm', 'yarn']}>
  <Tab value="pnpm">```bash pnpm create stackkit-app my-app ```</Tab>
  <Tab value="npm">```bash npx create-stackkit-app my-app ```</Tab>
  <Tab value="yarn">```bash yarn create stackkit-app my-app ```</Tab>
</Tabs>

## Choose Your Stack

The CLI will guide you through an interactive setup:

```bash
✔ What framework would you like? › Next.js
✔ Choose database › Prisma (PostgreSQL)
✔ Add authentication? › Better Auth
✔ Install dependencies? › Yes

✓ Project created successfully!
```

## Start Building

```bash
cd my-app
pnpm dev
```

Your app is running at [http://localhost:3000](http://localhost:3000)

## What's Included?

Your project comes with:

- ✅ **Framework** configured and ready
- ✅ **Database** client set up (if selected)
- ✅ **Authentication** configured (if selected)
- ✅ **TypeScript** strict mode enabled
- ✅ **ESLint** and **Prettier** configured
- ✅ **Environment variables** template
