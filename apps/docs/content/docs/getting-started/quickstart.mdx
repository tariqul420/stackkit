## Create a New Project

Run the interactive CLI to generate your project:

<Tabs items={['pnpm', 'npm', 'yarn', 'bun']}>
  <Tab value="pnpm">
    ```bash
    pnpm dlx stackkit@latest create my-app
    ```
  </Tab>
  <Tab value="npm">
    ```bash
    npx stackkit@latest create my-app
    ```
  </Tab>
  <Tab value="yarn">
    ```bash
    yarn dlx stackkit@latest create my-app
    ```
  </Tab>
  <Tab value="bun">
    ```bash
    bun x stackkit@latest create my-app
    ```
  </Tab>
</Tabs>

## Choose Your Stack

The CLI will guide you through setup:

```bash
✔ Project name: my-app
✔ Framework: Next.js
✔ Database: Prisma (PostgreSQL)
✔ Auth: Better Auth
✔ Language: TypeScript
✔ Package manager: pnpm

✓ Project created successfully!
```

**Framework Options:**
- **Next.js**: Full-stack React with App Router
- **Express**: Node.js REST API
- **React**: Frontend-only SPA

**Database Options:**
- **Prisma**: PostgreSQL, MongoDB, MySQL, SQLite
- **Mongoose**: MongoDB only
- **None**: Skip database setup

**Auth Options:**
- **Better Auth**: TypeScript-first, works with all frameworks
- **Auth.js**: Next.js + Prisma only
- **None**: Skip authentication

## Start Development

Navigate to your project and run the dev server:

```bash
cd my-app
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to see your app.

## Configure Environment Variables

StackKit generates `.env.example` with required variables. Copy it to `.env`:

```bash
cp .env.example .env
```

**For databases:**
```bash
# PostgreSQL
DATABASE_URL="postgresql://user:password@localhost:5432/myapp"

# MongoDB
DATABASE_URL="mongodb://localhost:27017/myapp"
```

**For Better Auth:**
```bash
BETTER_AUTH_SECRET="your-secret-here"  # Generate: openssl rand -base64 32
BETTER_AUTH_URL="http://localhost:3000"
```

<Callout type="warn">
Never commit `.env` files to version control.
</Callout>

## Adding to Existing Projects

Already have a project? Use the modular CLI:

<Tabs items={['pnpm', 'npm', 'yarn', 'bun']}>
  <Tab value="pnpm">
    ```bash
    # Add authentication (interactive)
    pnpm dlx stackkit@latest add

    # Add database with provider (non-interactive)
    pnpm dlx stackkit@latest add database --provider prisma-postgresql
    ```
  </Tab>
  <Tab value="npm">
    ```bash
    # Add authentication (interactive)
    npx stackkit@latest add

    # Add database with provider (non-interactive)
    npx stackkit@latest add database --provider prisma-postgresql
    ```
  </Tab>
  <Tab value="yarn">
    ```bash
    yarn dlx stackkit@latest add

    # Add database with provider (non-interactive)
    yarn dlx stackkit@latest add database --provider prisma-postgresql
    ```
  </Tab>
  <Tab value="bun">
    ```bash
    bun x stackkit@latest add

    # Add database with provider (non-interactive)
    bun x stackkit@latest add database --provider prisma-postgresql
    ```
  </Tab>
</Tabs>

See [CLI Reference](/docs/cli/overview) for all commands.
