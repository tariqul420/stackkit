## Requirements

StackKit requires Node.js 18 or higher.

**Recommended:**
- Node.js 20+
- pnpm (faster package manager)

**Supported package managers:**
- pnpm
- npm
- yarn
- bun

## Install StackKit

StackKit is distributed via npm. No separate installation needed.

```bash
# Verify Node.js version
node --version  # Should be 18+

# StackKit is ready to use
npx stackkit --help
```

## Project Structure

After creating a project, you'll have:

**Next.js:**
```
my-app/
├── app/          # App Router
├── lib/          # Utilities & clients
├── prisma/       # Database schema
├── .env          # Environment variables
└── package.json
```

**Express:**
```
my-app/
├── src/
│   ├── routes/   # API routes
│   ├── lib/      # Config & clients
│   └── server.ts
└── .env
```
