## Supported Stacks

### Frameworks

- **Next.js 16** (App Router) - Full-stack React framework
- **Express** - Minimal and flexible Node.js web framework
- **React + Vite** - Fast modern React development

### Authentication

- **Better Auth** - Modern authentication for TypeScript
- **Clerk** - Complete user management
- **Auth.js v5** (NextAuth) - Flexible authentication

### Databases

- **Prisma** (PostgreSQL/MongoDB) - Type-safe database access
- **Mongoose** (MongoDB) - ODM for MongoDB
- **Drizzle** (PostgreSQL) - SQL ORM

## Two CLI Tools

StackKit provides two complementary tools:

### create-stackkit-app

Create new projects with your chosen stack:

```bash
npx create-stackkit-app my-app
```

Select your framework, database, and auth during setup.

### stackkit-cli

Add modules to existing projects:

```bash
npx stackkit-cli add auth
npx stackkit-cli add database
```

Perfect for incrementally adopting StackKit features.
