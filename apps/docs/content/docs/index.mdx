# StackKit

Create production-ready applications with your preferred stack in seconds.

<Callout title="Quick Start">
  ```bash
  npx create-stackkit-app my-app
  ```
</Callout>

## What is StackKit?

StackKit is a CLI tool that generates fully-configured projects with:

- **Framework** - Next.js, Express, or React + Vite
- **Database** - Prisma or Mongoose with PostgreSQL/MongoDB
- **Authentication** - Better Auth or Clerk pre-configured
- **TypeScript** - Strict mode with best practices
- **Development tools** - ESLint, Prettier, and more

Stop wasting time on boilerplate. Start building features.

## Features

### Interactive Setup

Answer a few questions and get a fully configured project:

```bash
✔ What framework would you like? › Next.js
✔ Choose database › Prisma (PostgreSQL)
✔ Add authentication? › Better Auth
```

### Modular Design

Add features anytime without breaking existing code:

```bash
npx stackkit-cli add auth
npx stackkit-cli add database
```

### Production Ready

Every project includes:
- TypeScript strict mode
- ESLint and Prettier configured
- Database client set up
- Authentication flows ready
- Environment variables templated

## Supported Technologies

| Category | Options |
|----------|---------|
| **Frameworks** | Next.js 16, Express, React + Vite |
| **Databases** | Prisma (PostgreSQL/MongoDB), Mongoose (MongoDB) |
| **Authentication** | Better Auth, Clerk |
