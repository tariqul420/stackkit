StackKit generates full-stack projects and adds features to existing ones.

## Quick Start

<Tabs items={['pnpm', 'npm', 'yarn', 'bun']}>
  <Tab value="pnpm">
    ```bash
    pnpm create stackkit-app my-app
    ```
  </Tab>
  <Tab value="npm">
    ```bash
    npx create-stackkit-app my-app
    ```
  </Tab>
  <Tab value="yarn">
    ```bash
    yarn create stackkit-app my-app
    ```
  </Tab>
  <Tab value="bun">
    ```bash
    bun create stackkit-app my-app
    ```
  </Tab>
</Tabs>

## What it does

- Generates projects with Next.js, Express, or React
- Adds databases (Prisma, Mongoose) and auth (Better Auth, Auth.js)
- Handles dependencies, configs, and environment setup
