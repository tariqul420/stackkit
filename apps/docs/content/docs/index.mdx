## Features

- **Interactive CLI** - Choose framework, database, and auth in one command
- **Type-Safe** - Full TypeScript support with strict mode
- **Modular** - Add or remove features anytime using `stackkit-cli`
- **Production Ready** - Environment configs, error handling, and best practices included
- **Zero Config** - Everything works out of the box

## Quick Example

```bash
npx create-stackkit-app my-app
# ✔ Framework: Next.js
# ✔ Database: Prisma (PostgreSQL)  
# ✔ Authentication: Better Auth
# ✔ Install dependencies: Yes

cd my-app && npm run dev
```

Get a full-stack app with:
- Authentication routes and middleware
- Database schema and migrations
- API endpoints configured
- Environment variables templated
