## When to use

Using Prisma ORM with PostgreSQL, MongoDB, MySQL, or SQLite.

## Installation

### New project

<ModuleCreateCommands moduleType="database" moduleName="prisma" cliPrefix={"npx stackkit@latest create my-app"} />

### Existing project

```bash
# Interactive mode
npx stackkit@latest add

# Non-interactive mode
npx stackkit@latest add database --provider prisma-postgresql
```

## Providers

<ModuleProviders moduleType="database" moduleName="prisma" />

## What gets generated

- `prisma/schema.prisma`: Database schema
- `lib/db.ts`: Prisma client
- Migrations directory

## Setup

1. Set `DATABASE_URL` in `.env`
2. Run `npx prisma migrate dev`
3. Run `npx prisma generate`
