MongoDB object modeling for Node.js.

## Installation

### New project

```bash
npx stackkit@latest create my-app --database mongoose
```

### Existing project

```bash
npx stackkit@latest add
# Select "Database" â†’ "Mongoose"
```

## What gets generated

- `lib/db.ts` - MongoDB connection
- `lib/models/User.ts` - Example model
- Environment variable template

## Setup

1. Copy `.env.example` to `.env`
2. Set your `MONGODB_URI`
3. Ensure MongoDB is running (local) or accessible (Atlas)
4. Start dev server

Connection is handled automatically on app start.

## Verify MongoDB

```bash
# Local MongoDB
mongosh

# Check service (macOS)
brew services list | grep mongodb

# Check service (Linux)
systemctl status mongod
```

## Environment Variables

```bash
MONGODB_URI="mongodb://localhost:27017/mydb"

# For MongoDB Atlas
MONGODB_URI="mongodb+srv://user:pass@cluster.mongodb.net/mydb"
```
