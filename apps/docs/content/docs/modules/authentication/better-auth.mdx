TypeScript-first authentication for Next.js, Express, and React.

## Compatible Frameworks

- Next.js
- Express
- React

## Installation

### New project

```bash
npx stackkit@latest create my-app --auth better-auth
```

### Existing project

```bash
npx stackkit@latest add
# Select "Authentication" â†’ "Better Auth"
```

## What gets generated

- `lib/auth.ts` - Auth configuration
- API routes (framework-specific)
- Environment variable template

## Setup

1. Copy `.env.example` to `.env`
2. Generate secret:
   ```bash
   openssl rand -base64 32
   ```
3. Add your secret and URL to `.env`
4. Start dev server

## Environment Variables

```bash
BETTER_AUTH_SECRET="your-secret-key"
BETTER_AUTH_URL="http://localhost:3000"
```
