## When to use

Implementing authentication with Better Auth (TypeScript-first, open source).

## Installation

### New project

```bash
npx stackkit@latest create my-app --auth better-auth
```

### Existing project

```bash
npx stackkit@latest add
# Select "Authentication" â†’ "Better Auth"
```

## Compatible frameworks

<ModuleRequirements moduleType="auth" moduleName="better-auth" />

## What gets generated

- `lib/auth.ts`: Auth configuration
- API routes (Next.js)
- Environment variables in `.env.example`

## Setup

1. Set `BETTER_AUTH_SECRET` in `.env`
2. Set `BETTER_AUTH_URL` for production
