Auth.js integration for Next.js projects with Prisma database.

## Requirements

- Next.js framework
- Prisma database

## Installation

### New project

```bash
# PostgreSQL
npx stackkit@latest create my-app --framework nextjs --auth authjs --database prisma-postgresql

# MySQL
npx stackkit@latest create my-app --framework nextjs --auth authjs --database prisma-mysql

# SQLite
npx stackkit@latest create my-app --framework nextjs --auth authjs --database prisma-sqlite

# MongoDB
npx stackkit@latest create my-app --framework nextjs --auth authjs --database prisma-mongodb
```

### Existing project

```bash
npx stackkit@latest add
# Select "Authentication" â†’ "Auth.js"
```

## What gets generated

- `lib/auth.ts` - Auth configuration
- `app/api/auth/[...nextauth]/route.ts` - API routes
- Prisma schema updates
- Environment variable template

## Setup

1. Copy `.env.example` to `.env`
2. Generate secret:
   ```bash
   openssl rand -base64 32
   ```
3. Add your credentials to `.env`
4. Run Prisma migrations:
   ```bash
   npx prisma migrate dev
   npx prisma generate
   ```
5. Start dev server
