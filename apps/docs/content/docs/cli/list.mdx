## When to use

Discovering what modules are available for your project.

## Command

```bash
npx stackkit-cli list
```

## Output

Lists frameworks and modules grouped by category.

Example output:
```
FRAMEWORKS
  ├── Next.js
  ├── Express.js
  └── React (Vite)

MODULES
  ├── Auth
  │   ├── Better Auth
  │   └── Auth.js
  └── Database
      ├── Prisma (PostgreSQL, MongoDB, MySQL, SQLite)
      └── Mongoose (MongoDB)
```
