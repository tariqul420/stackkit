## Installation

No installation needed:

```bash
npx stackkit@latest <command>
```

<Callout type="info">
  Using `npx` ensures you always get the latest version.
</Callout>

## Commands

### create

Generate new project with framework, database, and auth.

```bash
stackkit create my-app
```

**What it does:**
- Creates project directory
- Copies framework template
- Integrates modules
- Installs dependencies
- Initializes Git

[Full docs →](/docs/cli/create)

### add

Add database or auth to existing project.

```bash
stackkit add
```

**What it does:**
- Detects your framework
- Shows compatible modules
- Generates integration code
- Installs dependencies

[Full docs →](/docs/cli/add)

### doctor

Verify project health.

```bash
stackkit doctor
```

**What it checks:**
- Required files exist
- Dependencies installed
- Configurations valid
- Environment variables documented

[Full docs →](/docs/cli/doctor)

### list

Show available options.

```bash
stackkit list
```

**Shows:**
- Supported frameworks
- Available modules
- Compatibility info

[Full docs →](/docs/cli/list)

## Global Options

| Option | Description |
|--------|-------------|
| `--help` | Show help |
| `--version` | Show version |
| `--verbose` | Detailed output |
| `--quiet` | Minimal output |

## Usage Examples

### Interactive (Recommended)

```bash
npx stackkit@latest create my-app
# Answer prompts
```

### Full Command

```bash
npx stackkit@latest create my-app \
  --framework nextjs \
  --database prisma-postgresql \
  --auth better-auth
```

### Minimal Start

```bash
# Start simple
stackkit create my-app --framework nextjs --database none

# Add features later
cd my-app
stackkit add
```
