StackKit provides a unified CLI for generating new projects and adding features to existing ones.

```bash
npx stackkit <command> [options]
```

### Commands

| Command | Description |
|---------|-------------|
| `create <name>` | Generate new project with framework + modules |
| `add <module>` | Add auth/database module to existing project |
| `doctor` | Check project health and compatibility |
| `list` | Show available frameworks and modules |

### Examples

```bash
# Create new project
npx stackkit create my-app

# Add authentication
npx stackkit add better-auth

# Add database with provider
npx stackkit add prisma --provider postgresql

# Check project setup
npx stackkit doctor

# List available options
npx stackkit list
```

### Common Workflows

1. **New project**: `npx stackkit create my-app`
2. **Add features**: `npx stackkit add better-auth`
3. **Verify setup**: `npx stackkit doctor`

See individual command pages for detailed options and examples.
