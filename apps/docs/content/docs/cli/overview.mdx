StackKit provides a unified CLI for generating new projects and adding features to existing ones.

```bash
npx stackkit@latest <command> [options]
```

### Commands

| Command | Description |
|---------|-------------|
| `create <name>` | Generate new project with framework + modules |
| `add <module>` | Add auth/database module to existing project |
| `doctor` | Check project health and compatibility |
| `list` | Show available frameworks and modules |

### Examples

```bash
# Create new project
npx stackkit@latest create my-app

# Add authentication (interactive)
npx stackkit@latest add

# Add database with provider (non-interactive)
npx stackkit@latest add database --provider prisma-postgresql

# Check project setup
npx stackkit@latest doctor

# List available options
npx stackkit@latest list
```

### Common Workflows

1. **New project**: `npx stackkit@latest create my-app`
2. **Add features**: `npx stackkit@latest add`
3. **Verify setup**: `npx stackkit@latest doctor`

See individual command pages for detailed options and examples.
