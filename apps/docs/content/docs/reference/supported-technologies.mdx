## Frameworks

| Framework | Status | Description |
|-----------|--------|-------------|
| Next.js | Stable | Full-stack with App Router |
| Express | Stable | Backend REST API |
| React | Stable | Frontend SPA with Vite |

## Databases

| Database | ORM/ODM | Providers | Frameworks | Status |
|----------|---------|-----------|------------|--------|
| Prisma | ORM | PostgreSQL, MongoDB, MySQL, SQLite | Express | Stable |
| Mongoose | ODM | MongoDB | Express | Stable |

## Authentication

| Provider | Frameworks | Databases | Status |
|----------|------------|-----------|--------|
| Better Auth | Next.js, Express, React | Prisma, Mongoose | Stable |

## UI

| Library | Frameworks | Status |
|---------|------------|--------|
| Shadcn UI | Next.js, React | Stable |

## Storage

| Provider | Frameworks | Status |
|----------|------------|--------|
| Cloudinary | Express | Stable |

## Components

| Module | Frameworks | Included | Status |
|--------|------------|----------|--------|
| Components | Next.js, React | TiptapEditor, DataTable, SearchBar, FilterBar, and more | Stable |

## Compatibility Matrix

| Module | Next.js | Express | React |
|--------|---------|---------|-------|
| Prisma | No | Yes | No |
| Mongoose | No | Yes | No |
| Better Auth | Yes | Yes | Yes |
| Shadcn UI | Yes | No | Yes |
| Cloudinary | No | Yes | No |
| Components | Yes | No | Yes |

## Notes

- React projects skip database selection (frontend-only)
- Better Auth on server frameworks (Express) requires a database
- Prisma is available for Next.js when added via Better Auth's schema setup
