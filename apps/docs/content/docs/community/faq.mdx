## General

**What is StackKit?**
A project generator and modular CLI for building full-stack apps, not a framework.

**Is it production-ready?**
Yes, all features are stable and tested.

## Usage

**Can I use StackKit on existing projects?**
Yes, use `npx stackkit add` to add features.

**What if I need unsupported tech?**
Contribute a module or use the generated code as starting point.

## Troubleshooting

**Doctor shows errors?**
Check environment variables and run migrations.

**Module conflicts?**
Use `--force` to overwrite, or check existing setup.
